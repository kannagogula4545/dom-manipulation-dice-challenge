# dom-manipulation-dice-challenge
"An interactive web application demonstrating DOM manipulation and asynchronous-style logic to create a dynamic two-player dice game."

## Features
- **Randomization:** Uses JavaScript's `Math.random()` to generate dice rolls.
- **Dynamic UI:** Updates the DOM in real-time to display different dice images and announce the winner.
- **Responsive Design:** Styled with CSS to ensure a clean look across different screen sizes.

## How to Play
1. Open the project in your browser.
2. Refresh the page to "roll" the dice.
3. The player with the higher number wins!

## Tech Stack
- **HTML5** for structure.
- **CSS3** for custom styling.
- **JavaScript** for game logic and DOM interaction.
